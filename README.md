Social Media App (Instagram Clone)

This project is a social media web application similar to Instagram where users can create posts, like posts, follow other users, and manage their profile. The backend is built using Django and Django REST Framework, and the frontend is developed with HTML, CSS, JavaScript, and Bootstrap.

Features

User registration and login with JWT authentication

Create posts with image and caption

View posts in the feed

Edit and delete posts

Like and unlike posts

Follow and unfollow users

User profile with followers and following count

Setup Instructions

Clone the repository

git clone https://github.com/yourusername/social-media-app-django.git

Navigate to the project folder

cd blog_project

Create a virtual environment

python -m venv venv

Activate the virtual environment

venv\Scripts\activate

Install dependencies

pip install -r requirements.txt

Run migrations

python manage.py migrate

Start the server

python manage.py runserver

Open the frontend pages such as login.html to use the application.
